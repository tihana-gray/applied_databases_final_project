# Applied Databases Final Project

## Overview
This project is a Python application developed as part of the Applied Databases module.

The application connects to:
- a MySQL database (`appdbproj`)
- a Neo4j database (`appdbprojNeo4j`)

and implements the functionality described in the project specification.

## Technologies Used
- Python 3
- MySQL (pymysql)
- Neo4j (Python Driver)
